# baby_monitoring_system
👶 Baby Monitoring System for Smart Cradle

📌 Overview

The Smart Cradle Baby Monitoring System is an IoT-based solution designed to ensure the safety and comfort of infants. The system continuously monitors the baby’s movement and environmental conditions such as temperature, and alerts caregivers in case of abnormal situations.

🚀 Features
Real-time baby movement detection using gyro sensor
Continuous temperature monitoring
Automatic alerts using buzzer for unsafe conditions
LCD display for live data visualization
Low-cost and efficient smart monitoring system
Easy to use and maintain
🛠️ Technologies Used
Microcontroller: NodeMCU (ESP8266)
Programming: Embedded C/C++ (Arduino IDE)
Sensors:
Gyro Sensor (Movement detection)
Temperature Sensor
Output Devices:
LCD Display
Buzzer
⚙️ System Architecture
Sensor Module → Collects data (movement & temperature)
Processing Module → NodeMCU processes data
Display Module → Shows real-time information
Alert Module → Triggers buzzer
Control Module → Manages system workflow
🔧 Implementation
Connect all hardware components to NodeMCU
Interface sensors for real-time data collection
Write and upload code using Arduino IDE
Display data on LCD screen
Trigger buzzer for abnormal conditions
Test and debug the system
📊 Working
Sensors continuously monitor baby movement and temperature
Data is sent to NodeMCU for processing
If values exceed safe limits → buzzer alert is activated
LCD displays current system status
System runs continuously in real-time
